# Configuration:

1. Télécharger Encrypt
2. Installer Python depuis le Store Windows
3. Dezipper Le fichier Encrypt.zip
4. Ouvrir votre ide ou le cmd dans la directory Encrypt
5. **Tapez la commande python main.py**
